Help and Documentation

Search Functionality

What you can do:


  1. Search for users via their username

  2. Search for media using tags, for example: love, nyc, etc.
    Note that while searching, you mustn't use the # symbol, just the tag is sufficient

  3. Search for media using an address or a location


On searching, you will see results pop up to your left. These results are accompanied by checkboxes. Use the checkboxes to select a result.


On selecting a result, you have the option to some easy access functions, such as add user to group. You can add the user to an already existing group or a new group, both for which a form will appear asking for further details.


If you're thrown an error, make sure to read it carefull and double-check that you have selected all the input methods required (for example, checkboxes on the result boxes, radio boxes for choosing an existing group, or even the radio buttons while searching for particular users/media)!


You also have the option to view a user's recent media and view a media's link on instagram, so feel free to do so. These options appear once you have search results present on your screen, within the individual result itself.

Manage Groups

What you can do:


  1. Create a new group

  2. Delete an existing group

  3. Delete a specific member from an existing group


In order to create a group, simply press the Create Group option from "Group Options". You have the option to give your group a customized name, else a name will be assigned to it for you.


To delete a group, you must have existing groups present on your interface. If you do so, select the group using the checkbox associated with it, go to group options, and press "Delete Group". All members associated with this group will automatically be deleted.


To delete a specific member of the group, press the delete option under the image of the member.


Remember that the groups are refreshed everytime you add/delete one of them, or even everytime you search, so everything is always up to date!
